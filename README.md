## 🚀 How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/Dhurkesh-B/LeetCode.git
   ```
2. Navigate to the problem folder:
   ```bash
   cd LeetCode/problem-number
   ```
3. Run the solution:
   ```bash
   python solution.py  # For Python solutions
   ```
